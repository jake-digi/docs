## List Subgroups

Retrieve all subgroups, optionally filtered by instance or parent supergroup.

<ParamField query={[
  { name: 'instance_id', type: 'string', description: 'Filter by instance ID (optional)' },
  { name: 'parent_supergroup_id', type: 'string', description: 'Filter by parent supergroup ID (optional)' }
]} />

### Example Request

```bash
curl -X GET 'https://api.checkpoint.b-digital.uk/api/v1/groups/subgroups?instance_id=se8SajpJDWUwrEmhElLF&parent_supergroup_id=supergroup123' \
  -H "X-API-Key: YOUR_API_KEY"
```
