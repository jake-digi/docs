## Create Instance

Create a new camp/session instance.

<ParamField body="name" type="string" required>
  Instance name
</ParamField>

<ParamField body="description" type="string">
  Instance description
</ParamField>

<ParamField body="status" type="string">
  Status: upcoming, active, completed, archived (default: upcoming)
</ParamField>

<ParamField body="start_date" type="string">
  Start date (YYYY-MM-DD)
</ParamField>

<ParamField body="end_date" type="string">
  End date (YYYY-MM-DD)
</ParamField>

<ParamField body="location" type="string">
  Location
</ParamField>

### Example Request

```bash
curl -X POST https://api.checkpoint.b-digital.uk/api/v1/instances \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Summer Camp 2025",
    "description": "Annual summer camp",
    "status": "upcoming",
    "start_date": "2025-07-01",
    "end_date": "2025-07-15"
  }'
```
